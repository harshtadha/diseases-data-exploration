This repository contains data exploring diseases and their trends.

## Getting Started

To get started with the data, clone the repository and install the necessary dependencies.

The dashboard tells multiple interconnected stories about the COVID-19 pandemic:

### 1. **Global Case Trends**
   - Tracks the rise and fall of COVID-19 cases across different countries and regions
   - Identifies pandemic waves and their timing across the globe
   - Shows how the virus spread from initial outbreak to worldwide pandemic

### 2. **Mortality Analysis**
   - Examines death rates across different countries
   - Compares case fatality rates to understand severity variations
   - Highlights regions most impacted by the pandemic

### 3. **Vaccination Impact**
   - Visualizes global vaccination progress over time
   - Demonstrates the correlation between vaccination rates and case reduction
   - Shows vaccination rollout timing across different countries

### 4. **Geographic Distribution**
   - Maps infection and death hotspots around the world
   - Identifies regional patterns and clusters
   - Compares performance across continents

### 5. **Daily Patterns & Trends**
   - Analyzes daily new cases and deaths
   - Identifies peak periods and turning points
   - Shows the trajectory of the pandemic over time

## Data Sources

- **CovidDeaths.xlsx** - Contains death and case statistics
- **CovidVaccinations.xlsx** - Contains vaccination data across countries

## Tools & Technologies

- **SQL** - Data exploration and analysis
- **Data Visualization** - Interactive dashboards for insights
